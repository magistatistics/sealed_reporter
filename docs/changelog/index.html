0.0.0-beta.2 (2025-03-10)

Evaluation improvements

  • Bomb CMC curve — Steeper penalty for high CMC in bomb score: 5-drops and 6-drops score lower, 7+ more negative, so expensive creatures (e.g. 7-mana bombs) don’t dominate rankings.
  • Removal saga penalty — Reduced the removal scorer’s penalty for subtype:saga (from -0.4 to -0.15) so removal sagas (e.g. The Last Ronin) aren’t over-penalized.
  • Bomb-heavy “all” composite — The overall card ranking (“all” tab) now weights bomb 2× and other categories 1×, so true bombs align better with draft tier lists.

Other

- Optional pandas — The scoring module (common/score) no longer requires pandas at import time when you only pass list-of-dicts (e.g. the CGB comparison script runs without installing pandas).

0.0.0-beta.1 — Beta launch

We're launching Sealed Reporter as a beta release.

What's in this release

  • Set reports — Browse and open set reports (e.g. TMT) with full dashboards and card signals.
