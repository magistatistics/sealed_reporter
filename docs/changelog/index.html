Beta launch

We're launching Sealed Reporter as a beta release.

What's in this release

  • Set reports — Browse and open set reports (e.g. TMT) with full dashboards and card signals.
  • Set info & B.R.E.A.D. — Overview, mana curve, type composition, and B.R.E.A.D. highlights.

  • Card exploration — Filter by category (Bomb, Removal, Evasion, Aggro, Diversity), color, and rarity.
  • Changelog — This page, loaded from markdown files.
  • Light / dark mode — Theme toggle in the header.
